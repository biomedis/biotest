You can see the button 1 in the upper panel; this button makes it possible to create the profile New profile in case of first launching. If you don’t create any profile, the button Start (Start measuring) will be inactive. If you want to edit parameters of profile, you need go to the chapter PROFILES (it will be presented below).

Then current active profile is displayed here; it can be replaced through pressing on button; choice of profile is memorized by program. All measurements take place in current active profile.

There are buttons rights on the same panel to go to the chapter of help, articles and work with profiles (it will be presented later).
Menu of results of current measurement is noticed through the number 2. Just the first item, corresponding to current screen, is active by default. After measurement is conducted, other items of menu will also become active for view of results. Results of measurement are displayed on the panel 3.
If you launch program without connecting pulse sensor, you will see the button Update status of USB-sensor (5). After connecting sensor, it is needed to inform program about it through pressing on the corresponding button. During the process of program’s work, you can remove sensor and connect it again; use the button 5 to start registration of data from sensor. If you have connected pulse sensor, and if the program records coming data, you will see live graph of pulse.
